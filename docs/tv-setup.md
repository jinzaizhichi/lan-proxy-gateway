# 智能电视网络设置指南

大多数智能电视（小米电视、Redmi 电视、海信、TCL、索尼 Android TV、飞利浦等）运行安卓系统，配置步骤和 Android 手机类似。

## 前提条件

- 电脑已运行 `sudo gateway start` 且显示启动成功
- 记下电脑的局域网 IP（启动时终端会显示，例如 `192.168.1.2`）
- 电视和电脑连接同一个 Wi-Fi / 路由器

## 设置步骤

### 1. 进入网络设置

不同品牌路径略有不同，常见路径：

```
设置 → 网络 → 无线网络（或 Wi-Fi）
```

部分电视在「系统设置」或「高级设置」里。

### 2. 进入 Wi-Fi 详细设置

点击已连接的 Wi-Fi 网络名称，找到「高级设置」或「修改网络」。

部分品牌需要**长按**遥控器 OK 键，或按遥控器**菜单键**来展开更多选项。

### 3. 设置静态 IP

将 IP 获取方式从「自动（DHCP）」改为「**手动**」或「**静态**」，填写：

| 字段 | 填什么 | 说明 |
|------|--------|------|
| IP 地址 | `192.168.x.105` | 同网段未使用的 IP（x 和电脑相同） |
| 子网掩码 | `255.255.255.0` | 保持默认 |
| 网关 | `192.168.x.2` | **电脑的 IP** |
| DNS 1 | `192.168.x.2` | **电脑的 IP** |
| DNS 2 | `8.8.8.8` | 可选，留空也行 |

> 例：电脑 IP 是 `192.168.1.2`，电视可设为 `192.168.1.105`，网关和 DNS 1 都填 `192.168.1.2`

### 4. 保存并重新连接

保存后断开 Wi-Fi 重新连接，使配置生效。

## 验证

打开 YouTube App 或其他流媒体应用，能正常播放就成功了。

## 各品牌电视操作说明

### 小米电视 / Redmi 电视

```
设置 → 网络 → 无线网络 → 点击已连接的 Wi-Fi → 高级设置 → IP 设置改为静态
```

如果找不到「高级设置」，尝试：
```
设置 → 更多设置 → 网络 → Wi-Fi → 长按当前连接网络
```

### 海信 / TCL 电视

```
菜单 → 设置 → 网络连接 → 无线网络 → 选中 Wi-Fi → 按遥控器设置键 → 高级设置
```

### 索尼 Android TV

```
设置 → 网络和互联网 → Wi-Fi → 点击已连接的网络 → 右上角铅笔图标（修改）
→ 展开「高级选项」→ IP 设置改为静态
```

### 飞利浦 Android TV

```
设置 → 无线和有线网络 → 连接到网络 → 选择你的 Wi-Fi
→ 高级设置 → 静态 IP
```

### Samsung 三星智能电视（Tizen 系统）

三星较新的电视使用 Tizen 系统，界面和安卓不同：

```
设置 → 通用 → 网络 → 网络状态 → IP 设置
→ IP 设置：手动
→ DNS 设置：手动
```

填写内容同上。

## 找不到静态 IP 设置怎么办

部分较旧的电视或品牌定制系统可能没有直观的「静态 IP」设置入口。可以尝试：

1. **搜索你的电视型号 + "手动设置 IP"**，找对应品牌的教程
2. **通过路由器绑定 IP**：在路由器后台找「DHCP 静态绑定」或「IP 与 MAC 绑定」，给电视的 MAC 地址绑定一个固定 IP，然后把该 IP 的网关和 DNS 设置在电视上填写（部分路由器支持直接在 DHCP 分配时推送自定义网关/DNS）

---

## 恢复自动网络

停止代理网关后，电视需要改回自动获取 IP，否则会断网：

```
设置 → 网络 → Wi-Fi → 选中当前网络 → 高级设置 → IP 设置改回 DHCP / 自动
```

或者：**忘记该 Wi-Fi 并重新连接**（最简单，重连后自动恢复 DHCP）。

## 常见问题

**电视上找不到高级网络设置**
不同品牌差异很大。通用方法：在 Wi-Fi 列表里长按你的网络名称，或按遥控器菜单键，通常能看到更多选项。实在找不到，搜索「[品牌型号] 手动设置 IP」。

**设置好了但视频还是看不了**
1. 确认电脑网关正在运行（`gateway status`）
2. 确认网关和 DNS 填的是电脑的 IP，而不是路由器的 IP
3. 在电脑浏览器打开管理面板 `http://电脑IP:9090/ui`，切换到速度更快的节点

**YouTube 能用，但 Netflix 打不开**
Netflix 需要支持解锁流媒体的节点。打开管理面板切换节点，找带「Netflix」、「流媒体」、「解锁」标签的节点。
