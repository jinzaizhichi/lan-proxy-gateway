# Nintendo Switch 网络设置指南

在电脑上启动 LAN Proxy Gateway 后，按以下步骤配置 Switch。

## 前提条件

- 电脑已运行 `sudo gateway start` 且显示启动成功
- 记下电脑的局域网 IP（启动时终端会显示，例如 `192.168.1.2`）
- Switch 和电脑连接同一个 Wi-Fi / 路由器

## 设置步骤

### 1. 进入网络设置

```
主页 → 设置 → 互联网 → 互联网设置
```

### 2. 选择 Wi-Fi

选择你的 Wi-Fi 网络 → **更改设置**

### 3. IP 地址设置

将 **IP 地址设置** 改为 **手动**：

| 设置项 | 填什么 | 说明 |
|--------|--------|------|
| IP 地址 | `192.168.x.100` | 同网段未使用的 IP（x 和电脑相同） |
| 子网掩码 | `255.255.255.0` | 保持默认 |
| 网关 | `192.168.x.2` | **电脑的 IP** |

> 例：电脑 IP 是 `192.168.1.2`，Switch 可设为 `192.168.1.100`，网关填 `192.168.1.2`

### 4. DNS 设置

**保持「自动」，不要修改 DNS。**

> 实测在 Switch 上手动配置 DNS 反而会导致连接异常（YouTube、eShop 打不开）。Switch 的网络实现比较特殊，只改网关即可，DNS 保持自动获取路由器分配的值就能正常使用。

### 5. 保存并测试

点击 **保存** → **连接到此网络**

测试连接应显示"已连接到互联网"。

## 验证

- eShop 能正常打开
- 在线游戏能正常连接
- YouTube App 能正常播放

## 关于 NAT 类型

通过网关代理后，Switch 的 NAT 类型通常显示为 B 或 C，这是正常现象，绝大多数在线游戏不受影响。如果特定游戏匹配有问题，尝试在管理面板（`http://电脑IP:9090/ui`）切换代理节点。

## 恢复自动网络

停止代理网关后，Switch 网络需要改回去，否则无法上网：

```
设置 → 互联网 → 互联网设置 → 你的 Wi-Fi → 更改设置
→ IP 地址设置：自动
→ DNS 设置：自动
```

或者更简单：删除该 Wi-Fi 连接并重新连接（会自动恢复 DHCP）。

## 常见问题

**连接测试失败**
1. 确认电脑上 `gateway status` 显示 mihomo 正在运行
2. 确认 Switch 的网关和 DNS 都填了电脑的 IP
3. 确认 IP 地址的前三段和电脑相同（如都是 `192.168.1.x`）

**能上网但 eShop 打不开**
分流规则可能需要更新。尝试 `sudo gateway update` 升级到最新版本，或在管理面板切换节点。

**NAT 类型显示 C 或 D**
TUN 模式下 NAT 类型可能变差，这属于正常现象。大部分在线游戏仍能正常运行。如对战有问题可在管理面板切换不同的代理节点。
