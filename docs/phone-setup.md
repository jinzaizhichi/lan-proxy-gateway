# 手机 / 平板网络设置指南

在电脑上启动 LAN Proxy Gateway 后，按以下步骤配置 iPhone 或 Android 手机 / 平板。

配好之后，手机**不需要安装任何代理 App** 也能科学上网。特别适合给家人的设备配置——他们完全无感，打开 Google、YouTube 就能直接用。

## 前提条件

- 电脑已运行 `sudo gateway start` 且显示启动成功
- 记下电脑的局域网 IP（启动时终端会显示，例如 `192.168.1.2`）
- 手机 / 平板和电脑连接同一个 Wi-Fi / 路由器

---

## iOS（iPhone / iPad）

### 1. 进入 Wi-Fi 设置

```
设置 → Wi-Fi
```

### 2. 进入网络详情

点击已连接网络名称右侧的 **① 按钮**（圆圈 i）。

### 3. 配置 IP

将 **配置 IP** 改为 **手动**：

| 字段 | 填什么 | 说明 |
|------|--------|------|
| IP 地址 | `192.168.x.100` | 同网段未使用的 IP（x 和电脑相同） |
| 子网掩码 | `255.255.255.0` | 保持默认 |
| 路由器 | `192.168.x.2` | **电脑的 IP** |

> 例：电脑 IP 是 `192.168.1.2`，iPhone 可设为 `192.168.1.100`，路由器填 `192.168.1.2`

### 4. 配置 DNS

在同一页面，将 **配置 DNS** 改为 **手动**：

- 删除现有的 DNS 服务器
- 点击「添加服务器」，填入**电脑的 IP**（如 `192.168.1.2`）

### 5. 保存

点击右上角 **存储**，完成配置。

### 验证

打开 Safari 访问 [google.com](https://google.com)，能正常加载就成功了。

## Android 手机 / 平板

> 不同品牌界面略有差异，以下以通用 Android 流程为准，华为 / 小米 / OPPO / 三星大同小异。

### 1. 进入 WLAN 设置

```
设置 → WLAN（或 Wi-Fi）
```

### 2. 修改网络配置

**长按**已连接的 Wi-Fi 网络名称 → 选择「**修改网络**」。

部分机型（如小米）直接点网络名称旁的设置箭头 / 齿轮图标。

### 3. 展开高级选项

勾选「**显示高级选项**」，或点击「高级」。

### 4. 改为静态 IP

将「IP 设置」从 DHCP 改为「**静态**」：

| 字段 | 填什么 | 说明 |
|------|--------|------|
| IP 地址 | `192.168.x.101` | 同网段未使用的 IP（x 和电脑相同） |
| 网关 | `192.168.x.2` | **电脑的 IP** |
| 网络前缀长度 | `24` | 即 255.255.255.0 |
| DNS 1 | `192.168.x.2` | **电脑的 IP** |
| DNS 2 | `8.8.8.8` | 可选，留空也行 |

> 例：电脑 IP 是 `192.168.1.2`，手机 IP 设为 `192.168.1.101`，网关和 DNS 1 都填 `192.168.1.2`

### 5. 保存

点击「保存」，完成配置。

### 验证

打开浏览器访问 [google.com](https://google.com)，能正常加载就成功了。

## 恢复自动网络

停止代理网关后，手机网络需要改回来，否则会断网：

**iOS：**
```
设置 → Wi-Fi → ① → 配置 IP → 自动 → 配置 DNS → 自动 → 存储
```

**Android：**
```
设置 → WLAN → 长按网络 → 修改网络 → IP 设置改回 DHCP → 保存
```

或者最简单的方式：**忘记该 Wi-Fi 并重新连接**，系统会自动恢复自动获取 IP。

## 常见问题

**部分 App 提示"检测到代理"**
TUN 模式是透明代理，通常不会被 App 检测到。如果个别 App 提示，一般不影响使用，可以忽略。

**国内 App 会变慢吗？**
不会。内置了智能分流规则，国内流量直连，只有访问国外网站才走代理，国内速度和以前完全一样。

**配置后完全无法上网（连国内网站也打不开）**
1. 运行 `gateway status` 确认网关正在运行
2. 确认手机的网关和 DNS 都填了电脑的 IP
3. 确认 IP 地址的前三段和电脑相同（如都是 `192.168.1.x`）
4. 尝试关闭 Wi-Fi 再重新打开

**家人的手机怎么配最省心？**
配好一次后，只要电脑网关一直开着，手机就一直能用，家人完全无感。唯一注意：电脑关机后，家人的手机需要改回自动获取 IP，否则会断网。建议安装系统服务（`sudo gateway service install`）让网关开机自启，这样只要电脑开着就不用管了。
