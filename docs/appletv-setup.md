# Apple TV 网络设置指南

在电脑上启动 LAN Proxy Gateway 后，按以下步骤配置 Apple TV。

## 前提条件

- 电脑已运行 `sudo gateway start` 且显示启动成功
- 记下电脑的局域网 IP（启动时终端会显示，例如 `192.168.1.2`）
- Apple TV 和电脑连接同一个 Wi-Fi / 路由器

## 设置步骤

### 1. 进入网络设置

```
设置 → 网络 → Wi-Fi
```

### 2. 选择 Wi-Fi 网络

点击当前连接的 Wi-Fi 网络名称，进入详情页。

### 3. 配置 IP

找到「配置 IP」，改为「**手动**」：

| 字段 | 填什么 | 说明 |
|------|--------|------|
| IP 地址 | `192.168.x.104` | 同网段未使用的 IP（x 和电脑相同） |
| 子网掩码 | `255.255.255.0` | 保持默认 |
| 路由器 | `192.168.x.2` | **电脑的 IP** |

> 例：电脑 IP 是 `192.168.1.2`，Apple TV 可设为 `192.168.1.104`，路由器填 `192.168.1.2`

### 4. 配置 DNS

找到「配置 DNS」，改为「**手动**」：

- 将 DNS 地址改为**电脑的 IP**（如 `192.168.1.2`）
- 点击已有地址进行修改，或删除后重新添加

### 5. 完成

返回上一级，Apple TV 会自动应用新的网络配置。

## 验证

- 打开 YouTube App，播放一个视频，确认能正常加载
- 打开 Netflix App，确认能浏览和播放内容
- 打开 Disney+ 或其他流媒体 App 测试

## 关于 Netflix

Netflix 对代理检测较严格，并非所有代理节点都能使用。如果 Netflix 提示地区错误或无法播放：

1. 打开管理面板：`http://电脑IP:9090/ui`
2. 在节点列表里找带有「解锁」、「流媒体」、「Netflix」标签的节点
3. 切换后重新打开 Netflix App 尝试

不同机场提供的节点解锁情况不同，可能需要多换几个节点。

## 关于 4K 播放

4K 视频需要稳定的高带宽（建议 25Mbps 以上）。如果 4K 播放卡顿或转圈，说明当前节点带宽不足，在管理面板切换到速度更快的节点。

## 恢复自动网络

停止代理网关后，Apple TV 网络需要改回来，否则会断网：

```
设置 → 网络 → Wi-Fi → 选择你的网络
→ 配置 IP：自动
→ 配置 DNS：自动
```

## 常见问题

**Netflix 提示"检测到代理"或地区错误**
需要切换到支持解锁流媒体的节点。打开管理面板 `http://电脑IP:9090/ui`，切换节点后重新尝试。

**4K 播放卡顿**
代理节点带宽不够。在管理面板中切换延迟更低、带宽更大的节点。

**连不上网**
1. 运行 `gateway status` 确认网关正在运行
2. 确认 Apple TV 的路由器和 DNS 都填了电脑的 IP
3. 确认 IP 地址的前三段和电脑相同（如都是 `192.168.1.x`）
4. 在 Apple TV 上重启网络：设置 → 网络 → Wi-Fi → 关闭再打开
