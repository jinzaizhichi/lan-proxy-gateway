MODE

接入能力

TUN 旁路由与 HTTP/SOCKS5 代理端口可同时提供。

透明代理 TUN 设备改默认网关和 DNS 即接入。适合投影仪、Switch、AppleTV。
代理服务 HTTP / SOCKS5 mixed-port 持续提供，设备也可以显式填写代理地址。
了解两种接入方式
透明代理（TUN）
内核态虚拟网卡接管 LAN 转发流量，适合只能改默认网关的设备。macOS 低干扰策略不会抢宿主 DNS。
代理服务（HTTP/SOCKS5）
仅监听 mixed-port，宿主机网络栈不动，设备需手动配置代理。
关系
它们不是互斥模式：TUN 开启时，HTTP/SOCKS5 代理端口仍然可用。

增强脚本

用 mihomo 配置后处理脚本（goja）加挂额外代理逻辑。预设可一键串住宅 IP 链路解锁 AI 站点。

把订阅里的机场节点合成「🛫 AI起飞节点」，加一个住宅 IP 包成「🏠 住宅IP」， 再用 mihomo 的 dialer-proxy 让住宅 IP 走机场出去 —— 对外看到的就是机场 → 住宅 IP，落地 IP 干净，解锁 ChatGPT / Claude / Gemini。

指向一个 .js 文件的绝对路径。文件内导出 function process(cfg) 对渲染后的 mihomo 配置做最终修改 —— 高阶用户用。

none 模式下保存即清空扩展脚本设置。
